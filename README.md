# Telegram Plugin for Claude Code

Marketplace hosting the Telegram messaging bridge plugin for Claude Code.

## Installation

```
/plugin marketplace add <owner>/<repo>
/plugin install telegram@telegram-plugin
```

See [`external_plugins/telegram/README.md`](./external_plugins/telegram/README.md) for setup.

## Structure

- `.claude-plugin/marketplace.json` — marketplace manifest
- `external_plugins/telegram/` — plugin source
